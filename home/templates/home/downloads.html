{{ page.question_one }}What is a Editon?

{{ page.answer_one }}A Editon is a diferent user Interface ( UI ), it changes the way you interact with you computer, try diferent images and find out what you like as it is purely personal preference, the base is still Manjaro.

{{ page.question_two }}Is Manjaro better that other distros?

{{ page.answer_two }}Manjaro is not superior or inferior to any other Linux distribution they all do the same, Manjaro has is own software, package managers themes Kernels and a rolling release model.

{{ page.question_three }}What about software availability?

{{ page.answer_three }}Manjaro allows installation of native packages, Snap, Flatpak or Appimage, since having a wide range of software is important, if your software of choice is not available you can try the AUR community repository.
